# PublicAPIWrapper Class

## Overview

The `PublicAPIWrapper` class provides a simplified interface to interact with various AI-related services. It handles authentication, request formatting, and response processing, making it easier to integrate these services into your applications.

## Constructor

### Parameters

- `apiKey`: Your unique API key for authentication
- `baseUrl`: Base URL of the API service (default: `'http://localhost:4333'`)

### Usage Example

```javascript
const apiWrapper = new PublicAPIWrapper('your-api-key');
```

## Methods

### imagine(options)

Performs image generation based on the provided options.

#### Parameters

- `options`: Object containing parameters for image generation

#### Returns

Promise resolving to the generated image data

#### Usage Example

```javascript
const result = await apiWrapper.imagine({
  prompt: 'Generate a cat',
  width: 512,
  height: 512,
});
```

### infer(RequestBody)

Performs large language model inference on the provided request body.

#### Parameters

- `RequestBody`: Object containing request parameters for LLM inference

#### Returns

Promise resolving to a readable stream of response data

#### Usage Example

```javascript
const stream = await apiWrapper.infer({
  model: 'gpt-3.5-turbo',
  messages: ['Start generating a short story'],
});
stream.on('data', (chunk) => {
  console.log(chunk.toString());
});
```

### search(options)

Performs text-based searches using the API.

#### Parameters

- `options`: Object containing parameters for the search operation

#### Returns

Promise resolving to the search results

#### Usage Example

```javascript
const searchResults = await apiWrapper.search({
  query: 'AI advancements in 2022',
  limit: 10,
});
console.log(searchResults);
```

### visualUpscale(options)

Enhances image resolution based on provided options.

#### Parameters

- `options`: Object containing parameters for image enhancement

#### Returns

Promise resolving to the enhanced image data

#### Usage Example

```javascript
const upscaleResult = await apiWrapper.visualUpscale({
  image: 'path/to/image.jpg',
  scale: 2,
});
```

### visualVariations(options)

Generates multiple variations of an image based on provided options.

#### Parameters

- `options`: Object containing parameters for image variation generation

#### Returns

Promise resolving to an array of generated image variations

#### Usage Example

```javascript
const variations = await apiWrapper.visualVariations({
  image: 'path/to/image.jpg',
  variations: 5,
});
```

### visualUpsample(options)

Upsamples an image to a higher resolution based on provided options.

#### Parameters

- `options`: Object containing parameters for image upscaling

#### Returns

Promise resolving to the upscaled image data

#### Usage Example

```javascript
const upsampleResult = await apiWrapper.visualUpsample({
  image: 'path/to/image.jpg',
  targetSize: { width: 1024, height: 768 },
});
```

### visualOutpaint(options)

Generates new content around an existing image based on provided options.

#### Parameters

- `options`: Object containing parameters for outpainting

#### Returns

Promise resolving to the generated outpainted image data

#### Usage Example

```javascript
const outpaintResult = await apiWrapper.visualOutpaint({
  image: 'path/to/image.jpg',
  prompt: 'Continue the scene with a futuristic cityscape',
});
```

## Error Handling

The class uses promises for asynchronous operations. Any errors during API calls will be rejected with a promise rejection. You should handle these rejections appropriately in your code.

## Notes

- Ensure that your API key is valid and has the necessary permissions for all operations you're performing.
- The base URL should match the actual endpoint of your API service.
- Some methods may have additional required fields or specific parameter formats. Always refer to the latest API documentation for the most accurate information.
